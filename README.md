# GPT-Minus
A simple C# WinForms app that lets you chat with multiple AI models. 🤖

## Requirements ⚙️
- Windows 10 or later (older versions not tested) 🖥️  
- [.NET Framework 4.7](https://dotnet.microsoft.com/en-us/download/dotnet-framework/net47) 🧩
